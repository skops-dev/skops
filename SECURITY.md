# Security Policy

## Supported Versions

| Version       | Supported          |
| ------------- | ------------------ |
| 0.11          | :white_check_mark: |
| < 0.11        | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities by opening a new [GitHub security
advisory](https://github.com/skops-dev/skops/security/advisories/new).
